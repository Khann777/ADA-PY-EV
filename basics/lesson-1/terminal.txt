Команды Linux / Terminal

ls - посмотреть все файлы/директории(папки) в текущей директории
ls test - показывает все файлы и папки внутри указанной директории 'test'
ls -a - показывает все файлы и директории, а также все скрытые файлы/директории
ls -l - показывает более подробную информацию о файлах и папках
ls -la - показывает более подробную информацию еще о скрытых файлах/папках

cd - переходит в домашнюю папку
cd "test" - переходи в заданную директорию (в нашем случае в test)
cd .. - переходит на одну директорию назад
cd ../.. - переходит на 2 директории назад (чтобы перейти еще дальше, можно через / указывать по 2 точки)

mkdir test - создает новую директорию test(именно в текущей директории)
mkdir test1/test2 - создает папку test2 в папке test1(если директория test1 не существует, то выйдет ошибка)
mkdir test1 test2 test3 test4 - создает 4 директории (test1-test4) именно в текущей папке

touch test.txt - команда, которая создает файл с любым расширешием в текущей директории
touch test1.py test1.txt test2.py test2.txt - создает много файлов с разными расширениями одновременно

rm test.txt - удаляет файл test.txt, если такого файла нет - выкинет ошибку, если указать директорию - также выкинет ошибку
rmdir test- удаляет директорию, удаляет только пустую, если в ней нет ни файлов ни папок
rm -rf test - удаляет файл или папку со всем содержимым (удаляет безвозвратно)

mv test.txt test1/ - перемещает файл test.txt в директорию test
mv test.txt test2.py - переименовывает файл test.txt в test2.py
mv test/ lesson1/ - переименовывает папку test в lesson1

history - показывает все команды, которые были выполнены в этом терминале
history 10 - показывет команды с 10 строчки и до конца
history 1 | grep 'cd' - показывает все команды с использованием слова cd с 1 строчки

pwd - показывает полный путь (начиная с корневой папки /) до текущей директории

nano test.txt - открывает файл test.txt в редакторе кода nano, если такого файла нет, то создает его
    Чтобы выйти из nano:
        1) ctrl + x
        2) y
        3) ENTER

cat test.txt - выводит в терминал все содержимое файла

code . - открывает редактор кода VSCode в текущей директории

sudo - команда, которые дает права суперюзера (super user do)
